									What is the HTML to hard-code a link to another site into a fragment?
								</h2>





							<div class="b-post__timestamp OLD__post-date"><time itemprop="dateCreated" datetime="2009-11-22T17:48:25">22-Nov-2009, 05:48 PM</time></div>

						</div>
					</div>


					<div class="OLD__post-content h-padding-vert-xl">

							<div class="axd axd_first-post">


								<!-- BEGIN: ad_first-post --><!-- END: ad_first-post -->
							</div>




						<div class="js-post__content-text OLD__post-content-text restore h-wordwrap" itemprop="text">


								The HTML to create a link is:<br />
<br />
&lt;a href=&quot;http://www.domain.co.uk/pagename.html&quot;&gt;Link Text Goes Here&lt;/a&gt;<br />
<br />
Replace 'www.domain.co.uk/pagename.html' with the URL that the link should go to and replace 'Link Text Goes Here' with the text to display for the link.  The above example would display as:<br />
<br />
<a href="http://www.domain.co.uk/pagename.html" target="_blank">Link Text Goes Here<br />
</a><br />
To use an image instead of text for the link replace 'Link Text Goes Here' with:<br />
<br />
&lt;img src=&quot;imagename.jpg&quot; alt=&quot;alt text&quot; border=&quot;0&quot;&gt;<br />
<br />
Replace 'imagename.jpg' with the name of your image and replace 'alt text' with the text to display if the mouse hovers over the image or if the image does not show for some reason.  You will need to add the image into the 'Design | Additional Files' list to ensure that it is uploaded to the website.<br />
<br />
To ensure that the HTML is correctly rendered when adding this into the fragment 'text' field you should surround the contents of the 'text' field with the embedding tags.  Put '!!&lt;' at the beginning and '&gt;!!' at the end (without the apostrophes).<br />
<br />
Alternatively, to have the link generated automatically, you can do the following:<br />
<br />
    * In Firefox, right click on your 'Bookmarks' toolbar, and click 'New bookmark'<br />
<br />
    * Give it a name of 'Link Generator'<br />
<br />
    * Paste the following code into the 'location' field:<br />
<br />
      javascript<b></b>:function htmlEscape(s){s=s.replace(/&amp;/g,'&amp;');s=s.replace(/&gt;/g,'&gt;');s=s.replace(/&lt;/g,'&lt;');return s;} function linkEscape(s){s=s.replace(/&amp;/g,'&amp;');s=s.replace(/&quot;/,'&quot;');return s} h = '&lt;a href=&quot;' + linkEscape(location.href) + '&quot;&gt;' + htmlEscape(document.title) + '&lt;/a&gt;'; with(window.open().document){write(h+'&lt;form name=f&gt;&lt;textarea  name=a rows=5 cols=80 wrap=hard&gt;'+htmlEscape(h)+'&lt;/textarea&gt;&lt;/form&gt;'); close(); f.a.select(); } void 0<br />
<br />
    * Click 'OK'<br />
<br />
    * Browse to your target page and click the bookmark<br />
<br />
    * This causes your browser to craft a link to the current page, using its title as the link text.  You can copy and paste this code into your fragment.
